API Reference#

Data types#

MarsTime is the primary data type of this package and represents a year and sol on Mars. MarsTimeDelta can be used in conjunction with MarsTime to create new instances of MarsTime. These may be useful on their own, but as with the datetime module these are probably most beneficial for attribute extraction.

MarsTime(year, sol)

A MarsTime represents the year and sol on Mars.

MarsTimeDelta([year, sol])

A MarsTimeDelta represents the difference in years and sols on Mars.

Conversion functions#

These function help converting between MarsTime and datetime.datetime objects. There are also functions for converting between sols and solar longitudes.

datetime_to_mars_time(dt)

Convert a datetime.datetime to a MarsTime.

mars_time_to_datetime(mt)

Convert a MarsTime to a datetime.datetime.

get_current_mars_time()

Get the current MarsTime.

solar_longitude_to_sol(solar_longitude)

Convert solar longitude to sol.

sol_to_solar_longitude(sol)

Convert sol to solar longitude.

Constants#

These constants are related to the orbit and/or timekeeping on Mars.

mars_year_0_start

Time of the start of Mars year 0.

hours_per_sol

Length of a Martian sol [hours].

seconds_per_sol

Number of seconds per Martian sol.

sols_per_martian_year

Number of sols per Martian year.

northern_spring_equinox_sol

The sol of the northern spring equinox (solar longitude = 0\(^\circ\)).

northern_summer_solstice_sol

The sol of the northern summer solstice (solar longitude = 90\(^\circ\)).

northern_autumn_equinox_sol

The sol of the northern autumn equinox (solar longitude = 180\(^\circ\)).

northern_winter_solstice_sol

The sol of the northern winter solstice (solar longitude = 270\(^\circ\)).

perihelion_sol

Sol of perihelion.

orbital_eccentricity

Mars' orbital eccentricity.
