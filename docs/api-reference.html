API reference¶

You can find all classes and functions present in the package at this page. In general, all classes have methods to convert to all other time representations.

EarthDatetime¶

class mer.EarthDatetime(dt)¶

An object that can convert Earth datetimes into Martian times.

Parameters

dt (datetime.datetime) – Any datetime.

Raises

TypeError – Raised if dt is not a datetime.datetime.

Notes

dt can be “aware” (have time zone information included with it) or “native” (have no associated time zone information). If it is native, it is assumed to be a UTC time and will have that info added to it.

Examples

This object simply accepts datetimes. You can include UTC time zone information as shown below.

>>> import datetime, mer
>>> utc = datetime.timezone.utc
>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0, tzinfo=utc)
>>> aware_dt = mer.EarthDatetime(dt)
>>> aware_dt
2020-01-01 00:00:00+00:00

If no time zone information is included, the datetime is assumed to be in UTC.

>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0)
>>> native_dt = mer.EarthDatetime(dt)
>>> native_dt
2020-01-01 00:00:00+00:00
>>> aware_dt == native_dt
True

This class can accept non-UTC timezones as well. I recommend using the pytz module if you want to use them.

>>> import pytz
>>> eastern = pytz.timezone('US/Eastern')
>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0, tzinfo=eastern)
>>> mer.EarthDatetime(dt)
2020-01-01 00:00:00-04:56

to_sol()¶

Compute the corresponding sol (day of the Martian year).

Notes

This function begins counting from 0. Beware that some places/people use the convention that the new year starts on sol 1.

Examples

Convert a datetime into its corresponding sol.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0, 0)
>>> mer.EarthDatetime(date).to_sol()
275.86418326244836

Beware that sols start from 0, or rather beware of hooligans counting from 1.

>>> mer.EarthDatetime(mer.mars_year_0_start).to_sol()
0.0
Return type

float

to_solar_longitude()¶

Compute the corresponding Martian solar longitude.

Examples

Convert a datetime into its corresponding solar longitude.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0, 0)
>>> mer.EarthDatetime(date).to_solar_longitude()
128.855367761636

References

The equation used in this method can be found in this paper. While this paper claims the equation is accurate to within 0.05 degrees, I have found errors up to 0.2 $^circ$.

Return type

float

Sol¶

class mer.Sol(mars_year, sol)¶

An object that can convert sols into different temporal representations.

Parameters

  • mars_year (int) – The Mars year.

  • sol (float) – The sol number, assuming the Mars year starts at sol 0.

Raises

  • TypeError – Raised if mars_year is not an int, or if sol is not an int or float.

  • ValueError – Raised if sol is an unphysical value.

to_datetime()¶

Compute the corresponding datetime.

Raises

sols_after_mars_year_0¶

mer.sols_after_mars_year_0(dt)¶

Compute the number of sols between a datetime and the start of Mars year 0.

Parameters

dt (datetime.datetime) – Any datetime.

Raises

TypeError – Raised if dt is not a datetime.datetime.

Return type

float

Examples

Find the number of sols after Mars year 0 that MAVEN arrived at Mars.

>>> import datetime, mer
>>> maven_arrival_datetime = datetime.datetime(2014, 9, 2, 2, 24, 0, 0)
>>> mer.sols_after_mars_year_0(maven_arrival_datetime)
21781.872772174716

sols_between_datetimes¶

mer.sols_between_datetimes(early_dt, later_dt)¶

Compute the number of sols between two datetimes.

Parameters

  • early_dt (datetime.datetime) – The earlier of the two datetimes.

  • later_dt (datetime.datetime) – The latter of the two datetimes.

Raises

TypeError – Raised if either early_dt or later_dt are not a datetime.datetime.

Return type

float

Examples

Compute the number of sols Opportunity was active. I don’t know the hour, minute, or second of the start or end of the mission so I set them to 0.

>>> import datetime, mer
>>> opportunity_start = datetime.datetime(2004, 1, 25, 0, 0, 0, 0)
>>> opportunity_end = datetime.datetime(2018, 6, 10, 0, 0, 0, 0)
>>> mer.sols_between_datetimes(opportunity_start, opportunity_end)
5109.551211085292

Constants¶

A collection of Martian time constants. Except where noted, these values come directly or indirectly from the Mars fact sheet.

Note that these can be accessed via the constants module (as shown) or used directly from mer’s namespace.
